# Bharat's Interview Prep Journey

A collection of UI challenges, polyfills, and JavaScript practice problems built with vanilla JS.

## Contents

- **polyfills** — custom implementations of debounce, throttle, memoize, flat, curry, call, bind, apply
- **autocomplete** — search autocomplete UI
- **infiniteScroll / infiniteScrolling** — infinite scroll implementation
- **movieSearch** — movie search UI
- **Search+Filter+Sorting** — table with search, filter and sort
- **table_component** — reusable table component
- **practice_solutions** — general JS practice
- **headout_interview** — Headout interview challenge
